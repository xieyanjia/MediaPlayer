# MediaPlayer

## 專案簡介

本程式主要提供基本影片播放功能的多媒體應用程式練習專案。程式使用 `OpenFileDialog` 讓使用者選取影片檔案，並透過 Windows Media Player 控制元件進行播放控制。

目前支援的主要功能包含：

- 瀏覽並選取本機影片檔案
- 播放影片
- 暫停影片
- 停止影片
- 影片靜音

---

## 功能說明

### 1. 瀏覽影片檔案

按下「瀏覽」按鈕後，會開啟檔案選擇視窗，讓使用者選擇想要播放的影片檔案。

選取成功後，程式會將影片路徑指定給播放器，並啟用播放、暫停與停止按鈕。

---

### 2. 播放影片

按下「播放」按鈕後，影片會開始播放。

---

### 3. 暫停影片

按下「暫停」按鈕後，影片會暫時停止在目前播放位置。

再次按下「播放」即可從暫停處繼續播放。

---

### 4. 影片靜音

按下「靜音」按鈕後，影片會靜音撥放。

---
